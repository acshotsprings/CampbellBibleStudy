Owner Mode — Quick Check

This page shows whether your current browser is flagged as the site owner. When owner mode is ON, analytics.js will NOT send you emails when you visit pages or save notes — it only emails you when a different visitor does those things.

Checking…

Activate owner mode on this device:

Click the button below to visit the site with ?owner=true. Do this once on every browser and every device you use to visit your own site (phone, tablet, laptop, work computer, etc.). The flag is stored per-browser in localStorage.

✓ Turn Owner Mode ON ✗ Turn Owner Mode OFF

What to expect:

  • After visiting the ON link, open the browser console (F12) — you'll see a log message confirming owner mode is enabled.
  • Visit any page on your site — you should NOT receive an email.
  • If a real visitor comes to the site from a different device, you WILL still get their email (that's the whole point).
  • Google Analytics tracking runs regardless, so you can still see your own visits in GA if you want. If you want to also exclude yourself from GA, that's a separate setting in GA's admin panel.

If you're still getting emails after turning owner mode ON:

  • Check that you visited the ?owner=true link on the same browser you're getting emails from.
  • If you use private/incognito browsing, the flag doesn't persist — avoid incognito for your own visits.
  • If you clear localStorage (or the browser does it automatically), you'll need to re-visit the ON link.
